还在 `↑ ↓` 一条条翻历史命令，或者 `Ctrl+R` 后瞪大眼在满屏乱码里找那条命令？

换个活法。装个 `fzf`，几十 KB 的 C 语言工具，装完你的终端像 IDE 一样带搜索框。

## 安装

```bash
# macOS
brew install fzf

# Linux
apt install fzf           # Debian/Ubuntu
dnf install fzf           # Fedora

# Windows
scoop install fzf         # 推荐
winget install fzf        # 也可
```

## 装完立刻起飞

加一行到 `.zshrc` 或 `.bashrc`：

```bash
source <(fzf --shell)
```

重开终端后按 `Ctrl+R`——惊喜来了。输入 `docker` 秒筛所有 `docker` 相关历史命令；输入 `deploy` 只显示部署相关的；上下选中直接回车执行。

这不是普通的搜索框，是**实时过滤 + 模糊匹配**。`dpl` 能匹配到 `deploy`。

## 三个实用场景

### 1️⃣ 文件搜索 + 打开

```bash
# vim 打开搜到的文件
vim $(fzf)

# 带预览：左边搜，右边看内容
fzf --preview 'bat --color=always {}'
```

### 2️⃣ 快速切换目录

```bash
# 交互式 cd
cd $(fd --type d | fzf)    # 配合 fd，快过 find
```

### 3️⃣ 搜索代码，选中打开

```bash
# 搜 "TODO" 的文件，实时预览，选中编辑
rg -l "TODO" | fzf --preview 'bat --color=always {}' | xargs nvim
```

## 让效果翻倍的组合

`fzf` 强在组合。配上这几个工具，终端体验不输 VS Code：

| 工具 | 作用 |
|------|------|
| **bat** | 语法高亮版 cat，预览代码必备 |
| **ripgrep (rg)** | 快出天际的 grep，搜代码用 |
| **fd** | 比 find 快 10 倍的查找工具 |
| **eza** | 彩色版 ls，配合 fzf 预览文件列表 |

一行配好默认预览：

```bash
echo 'export FZF_DEFAULT_OPTS="--preview \"bat --color=always {}\""' >> ~/.zshrc
```

## 为什么你该装

- **零依赖**：一个二进制 + 一个 `source` = 开箱即用
- **通用性**：任何管道数据都能喂给 fzf——搜索结果、git 分支、docker 容器、进程列表
- **无感**：装完不改变你任何习惯，只在需要搜索时才出现

花 3 分钟装上，之后每天省回来的时间远超 3 分钟。**这就是性价比最高的效率投资**。
