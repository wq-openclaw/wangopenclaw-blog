你还在 GitHub 网页上点点点搜代码？`gh search` 一行命令搞定，配合 `jq` 还能做批量操作。

## 0. 前提：装好 gh

```bash
# macOS
brew install gh

# Windows
winget install GitHub.cli

# 登录（一次）
gh auth login
```

## 5 个让你回不去的用法

### 1. 搜代码片段

```bash
# 在全网公开仓库搜索代码
gh search code "fetch('/api/users')" --language=js

# 限定仓库
gh search code "config.DatabaseURL" --repo=myorg/myapp

# 只看特定扩展名
gh search code "rate_limiter" --extension=py --extension=go
```

不用打开浏览器，终端里秒搜。

### 2. 搜 Issue/PR 并批量处理

```bash
# 搜你仓库里没人管的 issue
gh search issues --repo=myorg/myapp --state=open --label=none

# 搜别人写的，合并后批量关
gh search prs --author=me --state=open --review=none \
  --json=number --jq='.[].number' | \
  xargs -I{} gh pr close {} --comment "超时关闭，请重新提交"
```

以前要一个个点，现在一条管道搞定。

### 3. 搜仓库 + 装包

```bash
# 搜 star 上千的 React 模板
gh search repos "react starter template" --stars=">1000" \
  --sort=stars --limit=5

# 找到就 clone
gh search repos "tailwind blog template" --stars=">500" \
  --json=urlWithOwner --jq='.[].urlWithOwner' | \
  xargs gh repo clone
```

找轮子、clone 一步到位。

### 4. 搜自己的代码仓库

```bash
# 记不清某个配置在哪个私仓里？
gh search code "S3_BUCKET_NAME" --owner=myorg

# 搜所有仓库里过期的 TODO
gh search code "TODO: remove" --owner=myorg --extension=js \
  --json=path,repository --jq='.[] | "\(.repository.nameWithOwner): \(.path)"'
```

"我写过的 S3 配置放哪了？" → 一秒定位。

### 5. 搜历史 + 审计追踪

```bash
# 搜 commit 信息
gh search commits "fix security vuln" --repo=myorg/myapp

# 看谁引入了敏感信息
gh search code "password =" --owner=myorg --extension=env \
  --json=repository,path,textMatches

# 配合 blame 追源头
gh api repos/myorg/myapp/commits --jq='.[].sha' --paginate | \
  head -5 | xargs -I{} gh api repos/myorg/myapp/commits/{}
```

Code Review 找雷区、安全审计利器。

## 终极组合：gh + jq + fzf

```bash
# 交互式搜索 Issue，选中后直接打开
gh issue list --json=number,title --jq='.[] | "\(.number)\t\(.title)"' | \
  fzf | awk '{print $1}' | xargs gh issue view --web
```

## 一句话总结

> 学会 `gh search`，你就拥有了一个**终端版 GitHub 搜索引擎**。配合 jq 管道化输出，批量操作从"点半小时"变成"敲一行"。

用过的都说香，没用的下次搜代码试试？
