装了 fzf 之后，我再也回不去手动翻历史命令的日子了。用一句话概括：**任何需要"选"的场景，fzf 都能让你用模糊搜索一秒钟定位**。

## 安装

```bash
# macOS
brew install fzf && $(brew --prefix)/opt/fzf/install

# Linux (Debian/Ubuntu)
sudo apt install fzf

# Windows (scoop)
scoop install fzf
```

完了在 `~/.zshrc` 或 `~/.bashrc` 加一行：

```bash
[ -f ~/.fzf.zsh ] && source ~/.fzf.zsh
```

## 三个马上就能用的场景

### 1. 历史命令搜索（不用再按几十次 ↑）

默认 `Ctrl+R` 已经被 fzf 接管了。原本你得回忆"那条命令里到底用了什么参数"，现在敲几个关键词就模糊匹配出来，回车即用。

### 2. 文件搜索（替代 cd + ls 反复横跳）

按 `Ctrl+T` 就会弹出模糊搜索界面，选中的路径直接贴到命令行。搭配 `**` 补全也极其丝滑：

```bash
# 列出当前目录下的所有文件，模糊搜索选择
vim **  # Tab 触发 fzf 补全
cd **   # 选目录
kill -9 **  # 选 PID，用它杀进程比 ps aux | grep 快十倍
```

### 3. 预览模式

这是其他模糊搜索工具没有的杀招：

```bash
# 搜索文件时同时预览内容
fzf --preview 'bat {} --color=always' --preview-window 'right:60%'
```

配上 `bat` 或 `highlight`，搜索文件的同时右边直接显示文件内容预览，找配置文件时爽到飞起。

## 我的日常绑定

```bash
# 几个我每天都在用的 alias
alias fkill="ps -ef | fzf | awk '{print \$2}' | xargs kill -9"
alias fbr="git branch -a | fzf --height 40% | tr -d ' *' | pbcopy"
alias fcd="cd \$(find . -type d | fzf --preview 'ls -la {}')"
```

- `fkill` — 模糊搜索进程，选中的直接杀掉
- `fbr` — 搜 git 分支名，复制到剪贴板，省得手打
- `fcd` — 搜目录并跳转

## 就一句话

fzf 是那种**装上就觉得"我之前怎么没装"**的工具。花 3 分钟装一下，从今天开始你的每个 `Ctrl+R` 都会比以前快一个数量级。
