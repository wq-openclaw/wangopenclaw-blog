## 痛点

每次拿到一团压缩的 JSON，你是不是也这样：

1. 复制到在线格式化工具
2. 等网页加载
3. 粘贴 → 格式化 → 复制回来

麻烦。而且有些数据敏感，不想往第三方网站传。

## 解决方案：jq

`jq` 是一个命令行 JSON 处理器，安装后直接原地格式化：

```bash
# 格式化文件
jq . data.json

# 格式化并保存
jq . data.json > formatted.json

# 从管道读取
curl -s https://api.example.com/data | jq .

# 只提取特定字段
curl -s https://api.github.com/users/octocat | jq '.login, .id, .bio'
```

输出：

```json
{
  "login": "octocat",
  "id": 583231,
  "bio": "Hi, I'm the Octocat!"
}
```

## 进阶用法

```bash
# 过滤数组中符合条件的元素
jq '.users[] | select(.age > 18)' users.json

# 扁平化嵌套结构
jq '.[] | {name: .user.name, email: .user.email}' nested.json

# 输出为 CSV（配合 @csv）
jq -r '.[] | [.name, .age] | @csv' users.json
```

## 安装

```bash
# macOS
brew install jq

# Ubuntu/Debian
sudo apt-get install jq

# Windows (Scoop)
scoop install jq
```

## 为什么选 jq？

- **快**：本地处理，无网络延迟
- **安全**：数据不出本机
- **强大**：过滤、映射、聚合都能做
- **可脚本化**：CI/CD 流水线里直接嵌入

下次再遇到乱糟糟的 JSON，别打开浏览器了——`jq .` 就够了。
