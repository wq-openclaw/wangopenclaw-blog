## 还在用 `find` 和 `grep` 翻文件？换个组合拳，爽到飞起

每次在终端里找文件，你是不是还在：

```bash
find . -name "*.py" | xargs grep "something"
```

慢，还要等。尤其是项目大了以后，`find` 的每次遍历都像在教乌龟跑步。

今天介绍一个马上能用的组合：**fd + fzf + bat**。三分钟配置，搜索体验直接拉满。

### `fd`：带血的 `find` 替代

[`fd`](https://github.com/sharkdp/fd) 比 `find` 快 5-10 倍，语法还更友好：

```bash
# 递归查找所有 pdf 文件
fd pdf

# 查找包含 "config" 的文件名（大小写不敏感）
fd -i config

# 查找特定目录下的 .ts 文件
fd -e ts src/

# 查找并执行操作（配合 -x）
fd -e jpg -x convert {} {.}.png
```

默认忽略 `.gitignore` 里的文件，不会把 `node_modules` 里的垃圾翻出来。装完就能用：

```bash
# macOS
brew install fd

# Ubuntu 22.04+
sudo apt install fd-find

# Windows (scoop)
scoop install fd

# 其他系统
# https://github.com/sharkdp/fd#installation
```

### `fzf`：模糊搜索神器

[`fzf`](https://github.com/junegunn/fzf) 是通用模糊搜索工具。装完在你的终端里按 `Ctrl+R`，历史命令瞬间变成可搜索的列表。

把它和 `fd` 串起来才是大招：

```bash
# 交互式搜索文件名，选中后打开
fd -t f | fzf | xargs nvim

# 搜索并预览文件内容
fd -t f | fzf --preview 'bat --style=numbers --color=always {}'
```

配合 `bat` 的预览效果，搜索结果出来一屏显示文件内容，不用打开就知道是不是你要的。

### `bat`：带高亮的 `cat`

[`bat`](https://github.com/sharkdp/bat) = `cat` + 语法高亮 + 行号 + Git 改动标记。

```bash
# 带行号和高亮显示
bat config.json

# 只显示某几行
bat --line-range 10:30 app.py

# 在管道中只输出纯文本
bat --plain app.py | grep "TODO"
```

### 终极组合拳

把下面这行加到你的 `.zshrc` 或 `.bashrc`：

```bash
# 交互式文件搜索 + 预览 + 打开
function f() {
    local file
    file=$(fd -t f -H | fzf --preview 'bat --style=numbers --color=always {} 2>/dev/null' --height 60%)
    [[ -n "$file" ]] && nvim "$file"
}

# 搜索文件内容
function fgrep() {
    rg --line-number "$@" | fzf --delimiter : --preview 'bat --style=numbers --color=always --highlight-line {2} {1} 2>/dev/null' --preview-window +{2}-/2
}
```

有了这两个函数：
- 输入 `f` → 交互式选文件 → 自动用 neovim 打开
- 输入 `fgrep "TODO"` → 搜索所有含 TODO 的文件 → 一边预览一边选 → 直接跳转到对应行

全程键盘控制，不用离开终端，连鼠标都不需要碰。

### 一分钟安装

```bash
# macOS
brew install fd fzf bat

# Ubuntu
sudo apt install fd-find fzf bat  # 注意：Ubuntu 下 fd 命令叫 fdfind

# 装完加个 alias
echo "alias fd=fdfind" >> ~/.bashrc

# Windows (winget)
winget install sharkdp.fd junegunn.fzf sharkdp.bat
```

装好重启终端，从此找文件就像在 VS Code 的 Command Palette 里搜东西一样快。

### 效率不是靠毅力，是靠工具

这三个工具加起来不到 10MB，配置不到 10 行，但每天能给你省下几十次无意义的等待。高手和普通用户的区别往往不在知识量，而在**工具链的顺手程度**。

所以别再用 `find` + `grep` 受罪了，花三分钟装上这三个，你会回来感谢我的。
